# 第三章 列表
- 是什么：用[]来表示列表，用逗号分隔元素
  - 访问列表元素[n]，用元素的位置或索引
  - 索引从0而不是1开始
  - 使用列表的各个值
- 修改，添加和删除元素：列表是动态的
  - 修改：赋值
  - 添加：：xx.append()添加到末尾，xxx.insert(index,xs) 在列表中插入元素
  - 删除：del xxx[n],xx.pop()删除尾部元素，xxx.pop(n)弹出任何位置处的元素，xxx.remove()根据值删除元素
- 组织列表
  - sort() : 永久排序
  - sorted() : 临时排序
  -  反转列表： reverse()
  -  len() 长度
-  避免索引错误
